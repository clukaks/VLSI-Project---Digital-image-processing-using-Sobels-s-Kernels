# VLSI-Project---Digital-image-processing-using-Sobels-s-Kernels
FPGA Sobel Edge Detector | Arty Z7 Real-time 256x256 grayscale image processing at 115MHz.  VHDL pipeline with FIFO line buffers for 3x3 kernel.  Custom pipelined SQRT for gradient magnitude.  UART 115200 baud &amp; destructive RAM.  Bit-exact Python verification (scipy.signal.correlate2d). By: Luka Cirkovic &amp; Filip Derikonjic.
